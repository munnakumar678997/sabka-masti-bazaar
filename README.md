# 🎪 Sabka Masti Bazaar

> **Khelo • Jeeto • Kamao** — Desh ka sabse mast earning app! 🇮🇳

## 📱 App ke baare mein

**Sabka Masti Bazaar** ek desi earning web app hai jo bilkul mobile app jaisa dikhta aur feel karta hai. Yeh app Telegram ke saath tightly integrated hai — Mini App + Web dono pe kaam karta hai.

## ✨ Features (Abhi tak)

| Page | Status | Description |
|------|--------|-------------|
| 🎪 Loading / Splash Screen | ✅ Done | Animated splash with loading bar & floating coins |
| 🔐 Login Page | ✅ Done | Telegram OAuth — Mini App detect + Web widget |
| 🏠 Home Page | 🔜 Coming Soon | Balance, daily tasks, earning overview |
| 📅 Daily Check-in | 🔜 Coming Soon | Roz check-in karke coins kamao |

## 🛠️ Tech Stack

- **Frontend:** React 19 + Vite 8 + React Router DOM 7
- **Auth:** Telegram Login Widget + Mini App SDK
- **Backend:** Supabase (coming soon)
- **Deploy:** Firebase Hosting

## 📂 Folder Structure

```
src/
  pages/     → Har page ka alag JSX file
  styles/    → Har page ka alag CSS file
  assets/    → Images, icons
  App.jsx    → Routes
  main.jsx   → Entry point
```

## 🚀 Run karo

```bash
npm install
npm run dev
# App chal raha hai port 5000 pe
```
